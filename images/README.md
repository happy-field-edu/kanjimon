# カンジモン キャラクター画像フォルダ

このフォルダに各カンジモンの画像（PNG）を置くと、ゲーム内（孵化演出・図鑑・なかま棚）の絵文字が自動で画像に差し替わります。

- ファイル名は `index.html` の `MON_IMAGES` で指定済み（下表のとおり）。
- **正方形（1:1）・背景透過のPNG** 推奨。サイズの目安は 512×512px。
- 未設置のキャラは、これまでどおり絵文字で表示されます（混在OK・1体ずつ差し替え可）。
- 画像が読み込めない場合は自動で絵文字にフォールバックします。

# カンジモン 〜かんじ島クイズ〜（ベースシステム v0.1）

小学2年生向けの漢字学習ゲーム。「正しい漢字」と「よく間違える形の漢字」を
画像4択で見分けてカンジモンを集めるプロトタイプです。

## あそびかた
`index.html` をブラウザで開く（横画面推奨）。
URL欄を空のまま「サンプルもんだいで あそぶ」を押せばすぐ遊べます。

## 先生向け：問題の追加（Googleスプレッドシート）
1. スプレッドシートに以下の5列で問題を作成
   | 問題文 | 正解画像URL | 間違い画像URL1 | 間違い画像URL2 | 間違い画像URL3 |
   |---|---|---|---|---|
   | 「きいろ」の「き」 | https://…/ki_ok.png | https://…/ki_ng1.png | https://…/ki_ng2.png | https://…/ki_ng3.png |
2. メニュー「ファイル → 共有 → ウェブに公開」で **CSV形式** を選んで公開
3. 発行されたURL（`…/pub?output=csv`）をタイトル画面のURL欄に貼って開始

※ 1行目のヘッダー行は自動でスキップされます。
※ 形式の見本は `sample_questions.csv` を参照。
※ 問題文の「」内は画面で金色に強調表示されます。

## 実装済みのコア機能
- 公開CSVからの問題読み込み（失敗時はサンプル問題にフォールバック）
- 4択クイズUI（木の看板＋札を持つスライム4体、プニッと押せるボタン）
- タイムボーナスゲージ（10秒）／残り時間に応じたEXP
- コンボシステム（3コンボ以上で炎演出）／レベルアップ
- タマゴドロップ（基本22%＋コンボ×5%、最大70%）
  - 種類：しろ(3問)・あか(4問)・あお(5問)・にじ(7問)
  - コンボが高いほどレアなタマゴが出やすい
- ふらん器：正解するたびカウントダウン→孵化演出でカンジモン誕生
- 進捗の自動セーブ（localStorage：EXP・タマゴ・なかま）
- 効果音はWebAudioで生成（音声ファイル不要）
